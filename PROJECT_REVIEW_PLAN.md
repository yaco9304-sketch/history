## 1. 프로젝트 개요 및 목적

### 1.1 프로젝트 정보

| 항목 | 내용 |
|------|------|
| **프로젝트명** | 역사전쟁: 삼국시대 |
| **버전** | 0.0.1 |
| **설명** | 삼국시대(고구려, 백제, 신라)를 배경으로 한 실시간 멀티플레이 역사 교육 게임 |
| **코드 규모** | 약 13,162 라인 (서버 7개 파일, 클라이언트 26개 컴포넌트) |

### 1.2 최종 목표

**디지털교육연구대회 교육용 SW·AI 분과 최우수상 수상**

### 1.3 핵심 가치

1. **교육과정 연계**: 2015/2022 개정 교육과정 역사 교과 성취기준 달성
2. **학습 효과 측정**: 데이터 기반 학습 분석 및 리포트 제공
3. **교사 지원**: 수업 운영 및 평가에 실질적으로 도움이 되는 도구
4. **학생 참여**: 게임 기반 학습으로 높은 몰입도와 동기 부여

- Vite 5.4.10
- TailwindCSS 4.0.0
- Framer Motion 11.0.0

**Backend**
- Node.js + Express 4.18.2
- TypeScript 5.3.2
- Socket.io 4.7.2
- Google Generative AI 0.24.1 (Gemini API)

## 4. 문명 게임 참고 개선사항 제안

### 4.1 문명(Civilization) 게임의 핵심 요소

문명 시리즈는 턴제 전략 게임의 대표작으로, 다음과 같은 핵심 요소가 있습니다:

| 요소 | 설명 | 역사전쟁 적용 가능성 |
|------|------|---------------------|
| **테크 트리** | 기술 연구를 통한 문명 발전 | ⭐⭐⭐⭐⭐ 매우 높음 |
| **문화 승리** | 군사 외 다양한 승리 조건 | ⭐⭐⭐⭐ 높음 |
| **세계 불가사의** | 특별한 건물 건설 | ⭐⭐⭐⭐ 높음 |
| **위인 시스템** | 역사적 인물 등장 | ⭐⭐⭐⭐⭐ 매우 높음 |
| **지형 효과** | 지형별 전략적 특성 | ⭐⭐⭐ 중간 |
| **무역 시스템** | 국가 간 자원 교환 | ⭐⭐⭐⭐ 높음 |
| **시대 변화** | 문명 발전 단계 | ⭐⭐⭐⭐ 높음 |
| **도시 관리** | 세부적인 도시 운영 | ⭐⭐ 낮음 (복잡도 증가) |

### 4.2 구체적 개선 제안

```
├──────────────────────────────────────────────────────────┤

│  [농경 기술] ─→ [철기 기술] ─→ [제철 기술]               │
│       │              │              │                    │
│       ↓              ↓              ↓                    │
│  [관개 시설] ─→ [수공업 발달] ─→ [대장간 건설]           │
│                      │                                   │
│                      ↓                                   │
│              [불교 전파] ─→ [사찰 건립]                  │
│                      │                                   │

│              [문자 기록] ─→ [역사서 편찬]                │
│                                                          │
```

**교육적 효과**:
- 삼국시대 기술/문화 발전 순서 이해

**구현 난이도**: 중간 (1주 소요)

#### 🔥 우선순위 2: 위인 시스템 (강력 추천)

**현재 문제점**:
- 역사적 인물과의 연결 부족
- 게임과 실제 역사의 괴리

**개선안**:
```typescript
interface HistoricalHero {
  id: string;
  name: string;           // "광개토대왕", "근초고왕", "김유신"
  nation: Nation;
  era: string;            // "4세기 후반"
  specialAbility: {
    name: string;         // "정복왕의 기상"
    effect: string;       // "전투 승률 +20%"
    duration: number;     // 3턴
  };
  historicalFact: string; // 실제 역사적 업적
  unlockCondition: string; // 해금 조건
}
```

**위인 예시**:
| 국가 | 위인 | 특수 능력 | 역사적 사실 |
|------|------|----------|------------|
| 고구려 | 광개토대왕 | 정복왕의 기상: 전투 승률 +20% | 영토를 크게 확장 |
| 고구려 | 을지문덕 | 청야전술: 방어 시 적 피해 2배 | 살수대첩 승리 |
| 백제 | 근초고왕 | 해상왕국: 무역 수익 +30% | 해상 무역 발전 |
| 백제 | 계백 | 결사대: 최후의 항전 시 공격력 2배 | 황산벌 전투 |
| 신라 | 김유신 | 삼국통일: 동맹 체결 성공률 +50% | 삼국통일 주도 |
| 신라 | 선덕여왕 | 황룡사: 문화 발전 속도 +30% | 첨성대, 황룡사 건립 |

**교육적 효과**:
- 주요 역사 인물 자연스럽게 학습
- 인물의 업적과 시대적 배경 이해
- 역사적 사고력 향상

**구현 난이도**: 중간 (1주 소요)

```
무역 경로:
고구려 ←─[철, 가죽]─→ 백제
       ←─[말, 모피]─→ 신라

백제 ←─[곡물, 도자기]─→ 신라
     ←─[비단]─────→ 중국(이벤트)
     ←─[불경]────→ 일본(이벤트)
```

**기능**:
- 국가 간 자원 교환 제안
- 무역 협정 체결
- 무역 봉쇄 (전쟁 시)
- 실크로드/해상로 이벤트

**교육적 효과**:
- 삼국의 경제 활동 이해
- 국제 무역의 중요성 학습
- 협력과 경쟁의 균형

**구현 난이도**: 중간 (5일 소요)
