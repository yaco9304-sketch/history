## 대회 개요

### 디지털교육연구대회란?
한국교육학술정보원이 주최·주관하고 교육부가 후원하는 대회로, 에듀테크를 활용한 우수 교수·학습 실천 사례를 발굴하고 초·중·고 교실 수업 혁신과 미래지향적 수업 적용 확산을 통해 미래 인재 양성을 지원하기 위한 대회입니다.

### 대회 분과
- 인공지능(AI) 디지털교과서 활용 수업 운영 (신설)
- 디지털교수·학습
- **교육용 SW·AI** ← 우리 출품 분과
- 학교경영

## 출품 요건

### 필수 조건
- 출품자와 동일 학교급(유치원, 초등학교, 중등학교, 특수학교)이어야 함
- **교육과정과 연계**되고 교실 수업에 활용할 수 있는 소프트웨어(앱, 웹 등) 개발 및 적용
- 교육에 활용할 수 있는 AI, 빅데이터 프로그램 개발 및 적용

### 표절 주의사항 ⚠️
- **카피킬러 표절 검사 결과 20% 미만** 유지 필수
- 표절로 판명시 입상 취소 및 향후 3년간 모든 연구대회 출품 자격 제한

### 연구 절차
1. 개발하고자 하는 교과의 해당 내용 분석
2. 기 개발된 자료 분석
3. 개발 목표와 방향 설정
4. 필요에 따라 멀티미디어 자료(사진, 이미지, 소리, 동영상, 애니메이션 등) 활용

## 추가 필요 사항

### 1. 학습 효과 측정 시스템 ⭐⭐⭐⭐⭐ (최우선)

#### 현재 상태
- ❌ 학습자의 학습 성과 데이터 수집 및 분석 기능 없음
- ❌ 교사용 학습 데이터 대시보드 부재

#### 추가해야 할 기능

**학습 데이터 수집**
```
수집 항목:
- 게임 플레이 시간
- 의사결정 패턴 (어떤 선택을 했는지)
- 이벤트 선택 및 결과
- 외교/전투 활동 기록
- AI 어드바이저 활용 빈도
- 채팅 참여도 (협업 능력)
```

**학습 분석 대시보드**
- 교사가 학생들의 학습 현황을 한눈에 볼 수 있는 시각화
- 학급별/개인별 진도 현황
- 학습 목표 달성도 그래프
- 실시간 모니터링 기능

**학습 리포트 생성**
- 개인별 학습 성과 리포트 자동 생성 (PDF/출력 가능)
- 그룹별 협업 능력 분석
- 역사적 사고력 평가 리포트
- 강점/약점 분석 및 개선 방안 제시

**학습 목표 달성도 측정**
- 역사 지식 습득도
- 의사결정 능력
- 전략적 사고력
- 협업 및 소통 능력
- 문제해결력

### 2. 교육과정 연계 명시 ⭐⭐⭐⭐⭐ (필수)

#### 2015/2022 개정 교육과정 성취기준 매핑

**중학교 역사①**
```
성취기준: [9역01-03]
- 삼국 및 가야의 성립과 발전 과정을 파악하고,
  삼국 간의 항쟁 과정을 이해한다.

연계 내용:
- 고구려, 백제, 신라의 건국과 발전
- 삼국의 정치, 경제, 사회, 문화적 특징
- 삼국 간 외교 및 전쟁의 역사적 의미
```

**고등학교 한국사**
```
성취기준: [10한사01-02]
- 삼국의 형성과 발전 과정을 이해하고,
  고대 국가의 통치 체제와 대외 관계를 탐구한다.

연계 내용:
- 삼국의 중앙집권 국가 체제 확립
- 대외 교류와 문화 발전
- 역사적 사건의 원인과 결과 분석
```

#### 핵심 역량 연계
- **역사적 사고력**: 과거 사건의 원인과 결과 이해
- **비판적 사고력**: 역사적 의사결정의 타당성 평가
- **의사소통 역량**: 외교 채팅을 통한 협상 능력
- **공동체 역량**: 팀 채팅을 통한 협업
- **문제해결력**: 국가 위기 상황 대응 능력
- **창의적 사고력**: 다양한 전략 수립

### 3. 학습 피드백 시스템 ⭐⭐⭐⭐

#### 즉각적 피드백
```
예시:
학생 선택: "백제에 선제공격을 한다"
↓
시스템 피드백:
"475년 백제의 수도 한성이 고구려 장수왕에게 함락되었던
역사를 기억하세요. 선제공격은 국력 소모가 큽니다.
외교적 해결을 먼저 시도하는 것은 어떨까요?"
```

#### 맞춤형 학습 조언
- AI를 활용한 개인별 학습 가이드
- 플레이 패턴 분석 후 추천 전략 제시
- 약점 보완을 위한 학습 콘텐츠 추천

#### 오답 노트 기능
- 잘못된 의사결정 기록
- 역사적으로 올바른 선택 설명
- 재도전 기회 제공 (같은 상황 재현)

#### 성취 배지/보상 시스템
```
배지 예시:
- 🏛️ "평화 외교가": 전쟁 없이 30턴 생존
- ⚔️ "전략가": 5번 연속 전투 승리
- 📜 "역사학자": 모든 이벤트 경험
- 👑 "통일의 꿈": 삼국통일 달성
- 🤝 "협상의 달인": 외교 협정 10회 체결
```

### 5. 교육적 콘텐츠 강화 ⭐⭐⭐⭐

#### 역사 자료실
```
구성:
├── 삼국시대 연표
├── 주요 인물 소개 (광개토대왕, 근초고왕, 김춘추 등)
├── 역사 지도 (영토 변화)
├── 문화유산 소개 (고분, 사찰, 유물)
└── 역사 용어 사전
```

**각 이벤트/의사결정마다 연결**
- "이 사건에 대해 더 알아보기" 버튼
- 1차 사료 이미지 제공
- 역사학자의 해석 제시

#### 학습 가이드
- 게임 내 역사 용어 사전 (팝업/툴팁)
- 타임라인 시각화
- 삼국의 정치 체제 도표
- 주요 전투 지도

#### 토론 주제 제공
```
멀티플레이 종료 후 토론 주제:
1. "만약 내가 당시 왕이었다면 어떤 선택을 했을까?"
2. "삼국 중 어느 나라의 전략이 가장 효과적이었나?"
3. "역사적 사실과 게임의 선택이 다른 부분은?"
4. "현대 국제 관계에 적용할 수 있는 교훈은?"
```

#### 역사적 사고력 문제
- 단순 암기가 아닌 비판적 사고 유도
- "왜 그런 선택을 했나요?" 질문
- 역사적 맥락 분석 문제
- 다른 선택의 결과 예측 문제

### 7. 연구 자료 및 문서화 ⭐⭐⭐⭐⭐

**2. 교육적 효과 검증**
```
실제 수업 적용 사례:
- 적용 대상: OO중학교 2학년 3개 학급 (90명)
- 적용 기간: 4주 (8차시)
- 측정 도구: 사전-사후 검사, 설문조사, 인터뷰
- 결과 데이터:
  * 학습 동기 20% 증가
  * 역사 지식 이해도 35% 향상
  * 수업 만족도 4.5/5.0
  * 협업 능력 향상 긍정 응답 85%
```

```

【3-4차시】 외교와 전쟁

- 게임 활동: 멀티플레이어 협력/경쟁
- 정리: 역사적 사건과 게임 비교

【5-6차시】 문화와 발전
- 학습 목표: 삼국의 문화적 특징 파악
- 게임 활동: 문화 이벤트 중심 플레이
- 정리: 문화유산 조사 발표

【7-8차시】 종합 평가
- 학습 목표: 삼국시대 전반 이해 확인
- 게임 활동: 최종 경연 대회
- 평가: 학습 리포트 작성 및 상호평가
```

#### 수상작 특징 반영

**AI 활용 강화**
```
현재: ✅ AI 어드바이저 기본 기능 있음
```

**빅데이터 활용**
```
수집 데이터:

- 학습 효과가 높은 이벤트

활용 방안:
- 인기 전략 순위 제공

- 효과적인 수업 방법 도출
- 게임 밸런스 개선
```

**실생활 연계**
```
역사적 의사결정 → 현대 교훈 연결

예시:
게임 상황: "동맹국이 배신했다"
↓
현대 연결: "국제 관계에서 신뢰의 중요성"
         "계약과 조약의 법적 구속력"
         "외교 분쟁 해결 방법"
```

  - 로그 저장 기능 테스트

- [ ] **사용자 매뉴얼 작성**
