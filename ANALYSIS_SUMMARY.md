## ⚠️ 충돌 및 문제점

### 1. 타입 정의 중복 (경미)

**문제**:
```typescript
// 기존: types.ts
export interface EventHistory { ... }

// plan.md 제안
export interface LearningLog { ... }
```

**해결책**:
```typescript
// EventHistory 확장 또는 LearningLog로 통합
export interface LearningLog extends EventHistory {
  playerId: string;
  logType: 'event_choice' | 'battle' | 'diplomacy';
}
```

**영향**: 낮음 (1일 작업)

---

### 2. PDF 라이브러리 선택 (중요)

**plan.md 제안**: `pdfkit` (서버 측)

**문제점**:
- pdfkit은 Node.js 전용
- 클라이언트에서 미리보기 불가

**권장 변경**:
```bash
# 서버 측 대신
npm install pdfkit

# 클라이언트 측 사용 (더 간단)
npm install html2pdf.js
# 또는
npm install jspdf html2canvas
```

**이점**: 브라우저에서 직접 생성, 미리보기 가능

**영향**: 중간 (라이브러리만 변경, 로직 동일)
